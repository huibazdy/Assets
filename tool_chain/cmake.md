# CMake

【[CMake官网](https://cmake.org/)】

* **CMake是什么？**

    高级编译配置工具

    

* **什么情况下使用CMake？**

    多人用不同语言或不同编译器共同开发一个项目，最终输出一个可执行文件或共享库（dll或so）时

    

* **CMake的实现？**

    编译规则实现基于：`CMakeLists.txt`

    

* **为什么要学习CMake？**

    应对处理大型 C/C++/Java 项目

* CMake 最终输出的是什么？

    Makefile文件
