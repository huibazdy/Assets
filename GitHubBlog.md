# GitHub博客搭建

## 安装Hexo

### 准备工作

* 安装Git
* 安装NodeJS

### 安装Hexo

在终端输入命令：**`npm install -g hexo-cli`**

## 使用HEXO建站

1. 找一个盘符根目录，**`hexo init <blogfolder>`**
2. 切换到建立的blog路径，**`cd <blogfolder>`**
3. blog路径下初始化，**`npm install`**
